# Network-Knock-to-Unlock-Door-System
This is a project for my IoT class. The project is a home automation device that will be a part of a hub of devices that communicate and notify over a MQTT server. The door system and network system were programmed on different MCUs. A DHCP, TCP, and MQTT client were programmed from scratch on the first MCU. The door logic (knocking patterns/lock state machine) were programmed on the other MCU.
